Run the main file 'mainloop.m'. All the necessary functions are included
in 'wagLibrary.m'. When running the main file, you will be prompted to 
select the folder containing the data. The data are stored inside the 
folder 'data/pitch testing'. All the functions are thoroughly commented.

The final report including the results of the experiments is stored in the
'reports' folder. All the generated figures are also stored in the 'figures'
folder.
